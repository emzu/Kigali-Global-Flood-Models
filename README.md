# Kigali-Global-Flood-Models

Reliable flood models are critical for early warning systems. This work supports the report: "Enabling Urban Flood Models and Flood Early Warning Systems in Kigali city-Rwanda: A Review and Audit of Hydrometeorological Data and Models" with notebooks to analyze,  
1) Hydrology Station Data
2) HEC-HMS/HEC-RAS Model outputs  
3) Google Flood Hub: Real Time and Historical Datasets
4) GEOGloWS Flood Forecasts
